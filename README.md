# Algerian Forest Fire Prediction

A machine learning web app that predicts the Fire Weather Index (FWI) using a Ridge Regression model trained on the Algerian Forest Fires dataset.

## Inputs
- Temperature, Relative Humidity, Wind Speed, Rain
- FFMC, DMC, ISI indices
- Fire Class, Region

## Tech Stack
- Python, Flask, Scikit-learn
- HTML, CSS (custom dark theme)
